Color accent is a swap-point

Neutrals do the living. One working accent — Aubergine — defined at exactly one site; swap the token and every button, badge and bubble follows.

Shape & spacing

The big break from the old system: full-pill buttons and inputs, 16dp cards, uniformly rounded 18dp bubbles. Rounded, not rectangular — with pill CTAs and circular rosettes everywhere, rectangular bubbles would be the only sharp element on screen. Outgoing bubbles are solid accent with white text (Signal's move); incoming stay quiet gray. 8dp base grid, 24dp screen gutters, comfortable density.

The Rosette kept

Deterministic guilloché mark derived from each contact's MLS key fingerprint — banknote engraving, mathematically hard to forge. Verification engraves a second fine band (the one motion ceremony, 400ms). Everything else fades; nothing bounces.
