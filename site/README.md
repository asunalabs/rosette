# Rosette landing page

Next.js + TypeScript + Tailwind CSS v4, no backend. Design tokens and the
guilloché identicon algorithm are ported from `../DESIGN.md` and
`../docs/design/design-preview.html` — read those before changing colors,
typography, or `components/rosette.tsx`.

```bash
npm install
npm run dev     # http://localhost:3000
npm run build
```
